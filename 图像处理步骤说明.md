# 图像处理步骤说明

## 三个处理步骤

图像处理展示界面现在显示三个清晰的处理步骤：

### 步骤 1：原始图像
- **标签**：原始图像
- **图标**：📷 相机图标
- **说明**：全局视图
- **信息**：
  - 分辨率: 1920×1080
  - 大小: 2.4MB
- **边框颜色**：蓝色

### 步骤 2：图像分割
- **标签**：图像分割
- **图标**：⏳ 加载图标（动态旋转）
- **说明**：分割处理中...
- **信息**：
  - 算法: Watershed（分水岭算法）
  - 进度: 75%
- **边框颜色**：黄色
- **特效**：动态点点点动画

### 步骤 3：图像分类
- **标签**：图像分类
- **图标**：✅ 完成图标
- **说明**：分类完成
- **信息**：
  - 颗粒数: 1,245
  - 置信度: 96.8%
- **边框颜色**：绿色

## 处理流程

```
原始图像  →  图像分割  →  图像分类
   ↓           ↓           ↓
 采集图片    分离颗粒    识别分类
```

## 视觉效果

### 箭头指示
- 每个步骤之间有流动的箭头动画
- 箭头颜色：蓝色
- 动画效果：左右移动 + 透明度变化

### 卡片样式
- **原始图像**：蓝色边框，稳定显示
- **图像分割**：黄色边框，动态点点点，处理中状态
- **图像分类**：绿色边框，完成状态

### 悬浮效果
- 鼠标悬浮时卡片上升
- 边框发光效果增强
- 阴影加深

## 技术细节

### 算法说明
- **Watershed（分水岭算法）**：用于图像分割，将图像分成不同的区域
- **CNN-based**：基于卷积神经网络的分类算法

### 状态标识
- **原始图像**：`.image-box` - 基础样式
- **图像分割**：`.image-box.processing` - 处理中样式
- **图像分类**：`.image-box.result` - 结果样式
