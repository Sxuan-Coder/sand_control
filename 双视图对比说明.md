# 双视图图像处理对比说明

## 功能概述

图像处理展示界面现在包含**两套完整的处理对比**：
1. **全局视图** - 显示整体图像的处理流程
2. **局部视图** - 显示局部放大图像的处理流程

## 界面布局

```
┌─────────────────────────────────────────────────────────┐
│  图像处理对比                         [选择组别 ▼]      │
├─────────────────────────────────────────────────────────┤
│  全局视图                                                │
│  ┌──────────┐  →  ┌──────────┐  →  ┌──────────┐      │
│  │ 原始图像  │     │ 图像分割  │     │ 图像分类  │      │
│  │  全局    │     │  处理中   │     │  完成    │      │
│  └──────────┘     └──────────┘     └──────────┘      │
│                                                         │
│  局部视图                                                │
│  ┌──────────┐  →  ┌──────────┐  →  ┌──────────┐      │
│  │ 原始图像  │     │ 图像分割  │     │ 图像分类  │      │
│  │  局部    │     │  处理中   │     │  完成    │      │
│  └──────────┘     └──────────┘     └──────────┘      │
└─────────────────────────────────────────────────────────┘
```

## 全局视图 vs 局部视图

### 全局视图
- **用途**：显示整个样品的全景图像
- **分辨率**：1920×1080
- **文件大小**：2.4MB
- **颗粒数**：1,245（整体统计）
- **置信度**：96.8%
- **进度**：75%

### 局部视图
- **用途**：显示局部放大的细节图像
- **分辨率**：800×600
- **文件大小**：1.2MB
- **颗粒数**：856（局部统计）
- **置信度**：94.3%
- **进度**：82%

## 视觉标识

### 标签样式
```css
全局视图 / 局部视图
├─ 颜色：蓝色 (#00a8ff)
├─ 左边框：3px 蓝色实线
├─ 发光效果：文字阴影
└─ 动画：从左淡入
```

### 区分特点

| 特性 | 全局视图 | 局部视图 |
|------|---------|---------|
| 标签 | "全局视图" | "局部视图" |
| 占位符文字 | "全局视图" | "局部视图" |
| 分辨率 | 1920×1080 | 800×600 |
| 文件大小 | 2.4MB | 1.2MB |
| 颗粒数 | 1,245 | 856 |
| 置信度 | 96.8% | 94.3% |
| 处理进度 | 75% | 82% |

## 处理流程（两套相同）

每个视图都包含三个步骤：

### 步骤 1：原始图像
- 蓝色边框
- 相机图标
- 显示分辨率和文件大小

### 步骤 2：图像分割
- 黄色边框
- 加载图标（旋转动画）
- 动态点点点效果
- 显示算法（Watershed）和进度

### 步骤 3：图像分类
- 绿色边框
- 完成图标
- 显示颗粒数和置信度

## 动画效果

### 标签动画
- **fadeInLeft**：从左侧淡入
- 持续时间：0.5秒
- 缓动函数：ease-out

### 卡片动画
- 所有卡片保持原有动画
- 悬浮效果
- 箭头流动
- 点点点动画

## 间距设计

```
全局视图标签
  ↓ 15px
全局视图对比区域
  ↓ 30px
局部视图标签
  ↓ 15px
局部视图对比区域
```

## 使用场景

### 全局视图
- 查看整体分布情况
- 统计总体颗粒数量
- 评估整体处理效果
- 对比前后变化

### 局部视图
- 查看细节特征
- 验证分割准确性
- 检查分类效果
- 放大观察特定区域

## 响应式设计

- 两套对比区域垂直排列
- 保持相同的宽度和样式

## 优势

1. **对比清晰**：全局和局部同时展示，便于对比
